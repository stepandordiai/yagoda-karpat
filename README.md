# Yagoda Karpat

![Yagoda Karpat hero screenshot](./docs/screenshots/yagoda-karpat-ss-v9-c.png)

B2B website for an export-oriented company supplying frozen and dried agricultural products to international markets. The website highlights the product range, processing methods, quality assurance practices, and cooperation opportunities for distributors and wholesale partners.

## Tech stack

- Next.js (TypeScript)
- SASS
